# บันทึกการเปลี่ยนแปลง

บันทึกการเปลี่ยนแปลงสำคัญทั้งหมดของ Nostragoalus อยู่ที่นี่
รูปแบบเป็นไปตาม [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) โดยแต่ละเวอร์ชันเป็นสแนปช็อตตามวันที่ ไม่ใช่การปล่อยรุ่น

## [Unreleased]

## [2.5.0] - 2026-06-30

### เพิ่ม
- **ไม่พลาดการทายผล**: หน้าการแข่งขันตอนนี้แสดงจำนวนแมตช์ที่กำลังจะมาถึงซึ่งยังต้องการสกอร์ก่อนการล็อกครั้งถัดไป พร้อมปุ่ม "ไปที่แมตช์แรก" ที่เลื่อนตรงไปยังแมตช์ที่ยังไม่ได้ทายผลที่ใกล้ที่สุด
- **จับสกอร์ที่พิมพ์พลาด**: การกรอกสกอร์ที่สูงผิดปกติ (8 ประตูขึ้นไปต่อฝ่าย หรือ 12 ประตูขึ้นไปรวมกัน) ตอนนี้จะขอให้คุณยืนยันก่อนบันทึก เพื่อไม่ให้การพิมพ์พลาดอย่าง 1-33 ถูกล็อกไว้โดยไม่ตั้งใจ คุณยังยืนยันได้หากตั้งใจ
- **การเคลื่อนไหวของอัตราต่อรองและรายละเอียดต่อเจ้ามือ**: การแสดงอัตราต่อรองขนาดเล็กตอนนี้แสดงว่าแต่ละราคาขยับอย่างไรนับตั้งแต่เปิด - เครื่องหมายต่อผลลัพธ์สำหรับราคาที่ลดลง ขยายขึ้น หรือไม่เปลี่ยน พร้อมขนาดของการเคลื่อนไหว - และคุณแตะเพื่อขยายดูราคาเปิดและ 1X2 ต่อเจ้ามือได้ (เมื่อผู้ให้บริการมีข้อมูล) อัตราต่อรองแบบทศนิยมทั้งหมดเช่นเดิม
- **"การทายผลก่อนหน้าของคุณแม่นกว่า"**: ในแมตช์ที่คุณทายผลไว้ เมื่อแมตช์เริ่มแล้ว หน้าแมตช์จะบอกคุณได้เมื่อสกอร์ที่คุณเคยเลือกไว้ก่อนแล้วเปลี่ยนไปจะทำคะแนนได้ดีกว่าอันที่คุณเก็บไว้ - "1-0 ก่อนหน้าของคุณจะได้คะแนน อันที่คุณสลับออกไป" ระหว่างแมตช์สดจะเป็นค่าชั่วคราวและอัปเดตตามสกอร์ สรุปเมื่อจบเกม หากทายผลก่อนหน้าที่ชนะคือ 0-0 จะมีคำหยอกเล่นกว่า แสดงเฉพาะการทายผลเก่าของคุณเองให้คุณเท่านั้น
- **ดูว่าใครกำลังดูอยู่บ้าง**: หน้าแมตช์สดตอนนี้แสดง "มี N คนกำลังดูอยู่" - จำนวนแบบเรียลไทม์ว่ามีคนอยู่ในแมตช์นี้กี่คนในตอนนี้ อัปเดตเมื่อมีคนเข้ามาและออกไป
- **ตารางน็อกเอาต์แบบสด**: ตารางตอนนี้อัปเดตตามการแข่งขัน - สกอร์และการดวลจุดโทษเปลี่ยนแปลงในที่ พร้อมเครื่องหมาย LIVE บนคู่ที่กำลังแข่งอยู่ และผู้ชนะจะผ่านเข้ารอบต่อไปโดยไม่ต้องโหลดใหม่

### แก้ไข
- **ดวลจุดโทษแบบสด**: สกอร์การดวลจุดโทษใต้แมตช์ตอนนี้เพิ่มขึ้นตามการยิงแต่ละครั้ง ทั้งบนหน้าแมตช์และรายการแมตช์ แทนที่จะว่างเปล่าหรือค้างจนกว่าจะโหลดใหม่ สกอร์ในเวลาปกติด้านบนไม่เปลี่ยนแปลง
- **สกอร์ดวลจุดโทษในไทม์ไลน์**: การยิงจุดโทษที่เข้าแต่ละครั้งในไทม์ไลน์ play-by-play ตอนนี้แสดงผลรวมการดวลจุดโทษปัจจุบัน (1-0, 1-1, 2-1...) แทนที่จะแสดงสกอร์เวลาปกติที่ค้างอยู่ในทุกบรรทัด

## [2.4.0] - 2026-06-30

### เพิ่ม
- **กล่องข้อความแชทข้ามลีก**: รายการ "ห้องที่มีกิจกรรม" ในด็อกแชทตอนนี้รวบรวมห้องที่ยังไม่ได้อ่านจากทุกลีกที่คุณเป็นสมาชิก ไม่ใช่แค่ลีกที่คุณกำลังดูอยู่ และยังคงอยู่เมื่อโหลดหน้าใหม่ (ไม่รีเซ็ตเป็นว่างเปล่าเมื่อคุณรีเฟรชอีกต่อไป) การเลือกห้องจะกระโดดไปที่ห้องนั้นทันที พร้อมสลับลีกหากจำเป็น
- **การแจ้งเตือนเมื่อถูกกล่าวถึง**: การถูก @กล่าวถึงในแชทของลีกใดก็ตามตอนนี้จะส่งการแจ้งเตือนพุชเว็บและการแจ้งเตือนที่กระดิ่งส่วนหัวให้คุณ ไม่ว่าจะเป็นลีกหรือแมตช์ใด พร้อมลิงก์ตรงไปยังห้องนั้น เปิดใช้งานได้ในการตั้งค่า ในรายการการแจ้งเตือนพุช
- **เวลาเต็มเมื่อชี้เมาส์ค้าง**: การชี้เมาส์ค้างที่เวลาส่งของข้อความแชทตอนนี้จะแสดงวันที่และเวลาแบบเต็ม

## [2.3.0] - 2026-06-30

### เพิ่ม
- **บันทึกการเปลี่ยนแปลงพูดภาษาของคุณ**: บันทึกการปล่อยรุ่นในแอป (รายการ "มีอะไรใหม่" และหน้าเกี่ยวกับ) ตอนนี้อ่านในภาษาที่คุณเลือก โดยกลับไปใช้ภาษาอังกฤษสำหรับสิ่งที่ยังไม่ได้แปล แทนที่จะเป็นภาษาอังกฤษเสมอ
- **โปรไฟล์เปิดตรงจุดที่กำลังเกิดขึ้น**: โปรไฟล์ของผู้เล่นตอนนี้จะเลื่อนไปยังจุดที่ประวัติบรรจบกับปัจจุบัน - คำทายล่าสุดและคำทายสด - แทนที่จะเริ่มที่ด้านบนสุดของประวัติคำทายอันยาว

## [2.2.1] - 2026-06-30

### แก้ไข
- **การจับคู่สายน็อกเอาต์**: สายการแข่งตอนนี้เชื่อมแต่ละแมตช์รอบ 16 ทีม (และรอบถัดไป) เข้ากับแมตช์ต้นทางที่ถูกต้อง ก่อนหน้านี้ทุกการจับคู่อาจสลับผิด - เพราะถือว่าผู้ให้บริการเรียงแมตช์ตามลำดับหมายเลขแมตช์ ซึ่งฟีดฟุตบอลโลก 2026 ไม่ได้ทำ - ทำให้ผังแสดงทีมเจอกันผิดช่อง ตอนนี้ใช้หมายเลขแมตช์อย่างเป็นทางการของ FIFA แล้ว
- **ผู้ทำแอสซิสต์สูงสุดถูกต้อง**: กระดานแอสซิสต์ในแท็บ Stats ตอนนี้ตรงกับตัวเลขอย่างเป็นทางการ เดิมสร้างจากรายชื่อผู้ทำประตูสูงสุด (ผู้เล่นเพลย์เมกเกอร์ที่แอสซิสต์เยอะแต่ยิงน้อยจึงไม่เคยปรากฏ) และสำหรับฟุตบอลโลกที่กำลังแข่งอยู่ จะกลับไปใช้แอสซิสต์ที่ประมาณจากข้อมูลในเครื่อง ตอนนี้แอสซิสต์ถูกจัดอันดับแยกต่างหาก และระหว่างฟุตบอลโลก ดึงจากสถิติผู้เล่นสดอย่างเป็นทางการของ FIFA
- **ผู้เล่นที่เสมอกันใช้อันดับร่วมกัน**: ในกระดาน Stats ผู้เล่นที่มีจำนวนประตู (หรือแอสซิสต์) เท่ากันตอนนี้แสดงหมายเลขอันดับเดียวกัน โดยผู้เล่นคนถัดไปข้ามไปข้างหน้า (1, 2, 2, 2, 2, 6...) แทนที่จะถูกไล่หมายเลขตามอำเภอใจ

## [2.2.0] - 2026-06-28

### เพิ่ม
- **แท็บสถิติผู้เล่น**: มุมมองแมตช์มีแท็บ "Stats" ใหม่ถัดจาก Fixtures และ Standings แสดงผู้ทำประตูสูงสุดและผู้ทำแอสซิสต์สูงสุดของการแข่งขันเคียงข้างกัน มีที่ว่างให้ขยายไปสู่กระดานทีม (เกมรุก/เกมรับยอดเยี่ยม) ในภายหลัง

### เปลี่ยนแปลง
- **ปักหมุดอิมเมจของที่จัดเก็บอ็อบเจ็กต์**: อิมเมจ rustfs และ minio/mc ที่มาพร้อมกันถูกปักหมุดไว้กับ digest ที่ระบุเจาะจง ดังนั้นการสร้างใหม่หรือดึงสแตกใหม่จะไม่สลับไปใช้ที่จัดเก็บข้อมูลหรือไคลเอนต์สำรองข้อมูลอื่นอย่างเงียบ ๆ

## [2.1.0] - 2026-06-28

### เพิ่ม
- **ติดตั้งแอปได้จากในตัวแอป**: เมื่อเบราว์เซอร์เสนอให้ติดตั้ง Nostragoalus จะมีแบนเนอร์แสดงพร้อมปุ่มติดตั้ง แทนที่จะถูกซ่อนไว้ การปิดแบนเนอร์จะจดจำตัวเลือกของคุณ จึงไม่ถามซ้ำ

### เปลี่ยนแปลง
- **ขั้นตอนอัปเดตชัดเจนขึ้น**: แบนเนอร์เวอร์ชันใหม่จะแสดงขั้น "กำลังดาวน์โหลดอัปเดต" ขณะดึงบิลด์ถัดไป ดังนั้นคำเตือนให้รีโหลดจึงมาเป็นตอนจบของสิ่งที่มองเห็นได้ แทนที่จะโผล่ขึ้นมาเฉย ๆ บนมือถือจะอยู่เต็มความกว้างตามขอบล่าง แทนที่จะเป็นแถบแคบ ๆ ด้านบน

## [2.0.0] - 2026-06-28

### เพิ่ม
- **ที่จัดเก็บรูปภาพจริง**: รูปโปรไฟล์และรูปในแชทย้ายไปอยู่ในที่จัดเก็บที่ปรับตั้งได้ - เส้นทางไฟล์ระบบ หรือบริการที่รองรับ S3 ใด ๆ - แทนที่จะอยู่ในฐานข้อมูล สแตก Docker มาพร้อมบริการ S3 (rustfs) และตั้งให้แอปใช้งานเป็นค่าเริ่มต้น การย้ายข้อมูลครั้งเดียว (งาน `media:migrate-blobs` ในหน้างานเบื้องหลังของผู้ดูแล) จะย้ายรูปภาพเดิมออกจากฐานข้อมูล ให้รันจนกว่าทั้งสองจำนวนจะเป็นศูนย์

### เปลี่ยนแปลง
- **การสำรองข้อมูลรวมรูปภาพแล้ว**: `mise run db-backup` จะดัมป์ฐานข้อมูลและมิเรอร์ที่จัดเก็บรูปภาพไปพร้อมกัน จับคู่ด้วยเวลาประทับ และ `mise run db-restore` จะกู้คืนทั้งสองในครั้งเดียว (ใส่ `--no-media` เพื่อข้ามฝั่งรูปภาพในการดีพลอยแบบไฟล์ระบบเท่านั้น)

## [1.42.1] - 2026-06-28

### แก้ไข
- **ตัวอย่างลิงก์ทำงานอีกครั้ง**: การเปลี่ยนแปลงใน 1.42.0 ทำให้เซิร์ฟเวอร์ดึงตัวอย่างในแบบที่หลายเว็บ (เว็บที่อยู่หลัง Cloudflare เช่น 9gag) ปฏิเสธ ตัวอย่างจึงกลับมาว่างเปล่าอย่างเงียบ ๆ ตอนนี้โหลดได้อีกครั้ง และการป้องกันโฮสต์ส่วนตัว/ภายในยังคงอยู่

## [1.42.0] - 2026-06-28

### เพิ่ม
- **สถานะออนไลน์บนรูปโปรไฟล์**: จุดแสดงว่ามีใครอยู่หรือไม่ - เขียวเมื่อใช้งานอยู่ เหลืองอำพันเมื่อออนไลน์แต่ไม่ได้ใช้งาน ไม่มีจุดเมื่อออฟไลน์ - ทุกที่ที่รูปโปรไฟล์ปรากฏ
- **เธรด**: เปิดเธรดของข้อความเพื่อตอบกลับในบทสนทนาแยกที่เน้นเฉพาะ โดยไม่ปนกับห้องหลัก ลิงก์ "N การตอบกลับ" จะแสดงเมื่อเธรดมีความเคลื่อนไหว ซึ่งต่างจากการตอบกลับธรรมดาที่ยังอ้างข้อความในห้องแบบอินไลน์
- **การกล่าวถึง**: พิมพ์ `@` ในช่องเขียนข้อความเพื่อเลือกสมาชิกลีก - ชื่อจะแสดงขณะพิมพ์และแก้ไข ลิงก์ไปโปรไฟล์ และยังทำงานได้แม้สมาชิกจะเปลี่ยนชื่อภายหลัง เมื่อมีคนกล่าวถึงคุณ ฟองแชทที่ย่อไว้จะแสดงป้ายเฉพาะ (ยังนับรวมในยอดที่ยังไม่อ่านทั้งหมด) และเมนูห้องจะระบุว่าอยู่ห้องใด
- **ตัวเลือกอิโมจิ**: ปุ่มอิโมจิข้างปุ่มรูปภาพจะเปิดตัวเลือกที่ค้นหาได้และจัดหมวดหมู่ ซึ่งจะใส่อิโมจิลงในข้อความของคุณ
- **GIF**: วางลิงก์ GIF เพื่อแสดงแบบอินไลน์ (เคลื่อนไหว ไม่มีการจัดเก็บ) หรือแนบ GIF จากอุปกรณ์ของคุณแล้วยังคงเคลื่อนไหวได้
- **รูปภาพอินไลน์**: วางลิงก์รูปภาพแล้วจะแสดงในข้อความเลย - ย่อได้ ไม่มีการจัดเก็บ
- **ตัวอย่างลิงก์**: ลิงก์แรกในข้อความจะกางออกเป็นการ์ดที่ย่อได้ พร้อมชื่อหน้า คำอธิบาย และรูปภาพ
- **ค้นหาในแชท**: ปุ่มค้นหาจะกรองข้อความที่โหลดแล้วตามข้อความ (รวมถึงชื่อที่ถูกกล่าวถึง)
- **แก้ไขด้วยปุ่มลูกศร**: กดขึ้นในกล่องข้อความว่างเพื่อแก้ไขข้อความล่าสุดของคุณ กดขึ้น/ลงขณะแก้ไขเพื่อเลื่อนระหว่างข้อความของคุณ
- **จำกัดความยาวข้อความ**: ข้อความจำกัดที่ 2000 ตัวอักษร พร้อมตัวนับสดที่เตือนเมื่อใกล้ถึงและบล็อกการส่งเมื่อเกิน พร้อมอธิบายเหตุผล

### เปลี่ยนแปลง
- **ข้อความของคุณจะชิดขวา**ในแชท เหมือนแอปแชททั่วไป ส่วนของคนอื่นจะอยู่ทางซ้าย

- **รีแอกชันต่อข้อความ**: แสดงรีแอกชันต่อข้อความแชทใดก็ได้ด้วยอิโมจิ ชุดเดียวกับรีแอกชันของแมตช์ จำนวนอัปเดตแบบสดสำหรับทุกคน และการแตะรีแอกชันของตัวเองอีกครั้งจะลบออก
- **ตอบกลับข้อความ**: ตอบกลับข้อความใดก็ได้ แล้วข้อความต้นฉบับจะถูกอ้างไว้เหนือคำตอบของคุณ (และเหนือช่องเขียนข้อความขณะที่คุณเขียน) จึงติดตามบทสนทนาได้ง่าย ข้อความที่อ้างคลิกได้ - จะกระโดดไปและไฮไลต์โพสต์ต้นฉบับ ข้อความที่อ้างถูกถอดรหัสบนอุปกรณ์ของคุณเหมือนทุกอย่าง
- **รูปภาพในแชท**: ลากแล้ววาง วาง หรือแนบรูปภาพ (ไม่เกิน 5 MB) รูปจะถูกย่อเป็น webp และเข้ารหัสแบบต้นทางถึงปลายทางบนอุปกรณ์ของคุณก่อนส่ง ดังนั้นเว็บจึงเก็บแค่ข้อมูลที่อ่านไม่ออก - ไม่เคยเห็นรูป แตะรูปเพื่อขยาย แล้วคัดลอก ดาวน์โหลด หรือแชร์
- **แก้ไขและลบข้อความของคุณเอง**: แก้ข้อความหลังส่ง - จะแสดงเครื่องหมาย "แก้ไขแล้ว" พร้อมเวลา และข้อความใหม่จะถูกเข้ารหัสซ้ำบนอุปกรณ์ของคุณและอัปเดตแบบสดสำหรับทุกคน - หรือลบข้อความ ซึ่งจะแสดงเป็น "ข้อความถูกลบ" และคงการตอบกลับไว้ครบ คุณยังถอนรายงานที่คุณทำได้ด้วย
- **รายงานและกลั่นกรองข้อความแชท**: ใครก็ตามสามารถรายงานข้อความที่ตัวเองไม่ได้ส่งได้ เมื่อมีสมาชิกหลายคนรายงานข้อความหนึ่งมากพอ (หนึ่งในสี่ของลีก อย่างน้อยสามคน) ข้อความจะถูกซ่อนอัตโนมัติเป็น "รอตรวจสอบ" สำหรับทุกคนยกเว้นเจ้าของและผู้ดูแล เจ้าของและผู้ดูแลจะมีรายการรายงานเพื่ออ่านแต่ละข้อความที่ถูกรายงานและเลือกเก็บไว้หรือลบ และลบข้อความใดก็ได้โดยตรง ข้อความที่ถูกลบจะแสดงเป็น "ข้อความถูกลบ" และคงการตอบกลับไว้ การรายงานและการลบจะปรากฏแบบสดสำหรับทุกคน

### แก้ไข
- **การเข้าร่วมแชทจะไม่ทำให้คุณค้างอีกต่อไป**: สมาชิกที่เข้าร่วมลีกที่เปิดแชทอยู่แล้วเคยค้างอยู่ที่ "กำลังตั้งค่ากุญแจของคุณ..." ตลอดไป เว้นแต่จะมีคนที่อยู่ในแชทอยู่แล้วบังเอิญเปิดแชทขึ้นมา ตอนนี้อุปกรณ์ของคุณจะขอกุญแจจากกลุ่มโดยอัตโนมัติและปลดล็อกทันทีที่สมาชิกที่ถือกุญแจออนไลน์ พร้อมแสดงข้อความ "กำลังรอให้เข้าร่วม" จนกว่าจะถึงตอนนั้น
- **เลิกปิดเสียง**: การปิดเสียงสมาชิกเคยเป็นการถาวรโดยไม่มีทางย้อนกลับ ตอนนี้แชทมีรายการ "ปิดเสียงไว้" ที่คุณเลิกปิดเสียงใครก็ตามที่คุณซ่อนไว้ก่อนหน้านี้ได้
- **เปิด/ปิดแชทแบบสด**: การเปิด ปิด หรือหมุนกุญแจแชทจะอัปเดตสำหรับทุกคนทันที - หน้าต่างแชทจะปรากฏหรือหายไปสำหรับสมาชิกคนอื่นโดยไม่ต้องรีเฟรช แทนที่จะค้างเก่าจนกว่าพวกเขาจะรีโหลด
- **ชื่อของสมาชิกที่ซ่อนไว้ในแชท**: สมาชิกที่ซ่อนจากกระดานอันดับสาธารณะจะแสดงชื่อในแชทลีกที่ตัวเองอยู่ แทนที่จะปรากฏเป็น "ใครบางคน"
- **การเลื่อนแชท**: การรีแอกชัน ปิดเสียง หรือกลั่นกรองจะไม่ดึงมุมมองไปที่ข้อความล่าสุดอีกต่อไป ข้อความใหม่ขณะที่คุณเลื่อนขึ้นไปจะแสดงปุ่ม "ข้อความใหม่" ให้กระโดดไป แทนที่จะบังคับเลื่อนลง

### แก้ไข
- แผงการแจ้งเตือน (และเมนูบัญชี) จะปิดเมื่อคุณเลื่อนหน้า แทนที่จะลอยห่างจากส่วนหัวและทับตารางแข่งด้านล่าง
- แผนที่รายการแข่งจะวางธงให้ทุกชาติ ไม่ใช่แค่ขาประจำฟุตบอลโลกและยูโร: ทีมหน้าใหม่อย่างเคปเวิร์ดและคูราเซา (และทีมที่ผ่านเข้ารอบอื่น ๆ) เคยถูกตัดออกอย่างเงียบ ๆ ก่อนหน้านี้

## [1.17.0] - 2026-06-15

## [1.15.0] - 2026-06-15

### เพิ่ม

- คำถามที่พบบ่อยเรื่องการให้คะแนนตอนนี้ระบุโบนัสความหายากของผล (ส่วนเพิ่มเล็กน้อยสำหรับการทายผลที่หายากแต่ถูกต้อง) ที่มาพร้อมการอัปเดตการให้คะแนนแบบปรับได้ - ก่อนหน้านี้แสดงเฉพาะระดับความหายากของสกอร์ที่ตรงเป๊ะ

- **โบนัสความหายากของผล**: นอกเหนือจากความหายากของสกอร์ที่ตรงเป๊ะที่มีอยู่ ตอนนี้มีโบนัสเล็กน้อยให้รางวัลการทายผลที่หายากแต่ถูกต้อง (เช่น หนุนทีมรองเมื่อคนส่วนใหญ่ทายทีมเต็ง) เป็นชั้นเพิ่มที่ปรับได้บนโบนัสคนทาย เปิดเป็นค่าเริ่มต้นสำหรับการติดตั้งใหม่และตั้งค่าได้ต่อรายการแข่ง

### แก้ไข
- ในลีก การชี้ที่สกอร์คนทายใต้แมตช์จะไม่ซ้อนทูลทิปสองอันทับกันอีกต่อไป: ยอดรวมของลีกและของทุกคนจะแสดงป้ายกำกับใต้บรรทัด

## [1.11.0] - 2026-06-14

### เพิ่ม
- **โอกาสที่สองสำหรับคำทายแชมป์และรองเท้าทองคำ**: เมื่อรอบแบ่งกลุ่มใกล้จบ (ตั้งแต่รอบกลุ่มสุดท้ายจนถึงเริ่มรอบน็อคเอาท์) คุณจะได้โอกาสสุดท้ายในการเปลี่ยนคำทายผู้ชนะทัวร์นาเมนต์และดาวซัลโว - ได้แต้มครึ่งเดียว การเปลี่ยนเป็นการถาวร (เปลี่ยนกลับก็ไม่คืนมูลค่าเต็ม) มีคำยืนยันอธิบายชัดเจน และคำทายเดิมของคุณยังแสดงอยู่ข้างคำทายใหม่ ผู้เล่นที่ไม่เคยทายก็เลือกครั้งแรกแบบช้าในช่วงนี้ได้ ที่ครึ่งแต้ม

### แก้ไข
- โชว์เคสดาวซัลโวจะแสดงรูปหน้าทางการของผู้เล่นเมื่อมีอยู่เฉพาะในฟีดรายชื่อทีม (ผู้เล่นบางคน เช่น เอดิน เชโก ไม่มีรูปในเส้นทางที่อิงจาก id ที่เราใช้ และตกไปใช้ธงทีมแทน)
- บนมือถือ รายการตารางแข่งจะรอให้การ์ดแชมป์และดาวซัลโวด้านบนโหลดเสร็จก่อนจึงปรากฏ การกระโดดไปแมตช์ถัดไปตอนโหลดจึงไปลงที่แถวที่ถูกต้อง แทนที่จะหยุดก่อนถึงเมื่อการ์ดเหล่านั้นขยาย

- วันที่จะทำตามภาษาที่เลือกทุกที่ (ตารางแข่ง หน้าแมตช์ สายน็อคเอาท์ แผนที่ หน้าทีมและโปรไฟล์) แทนภาษาของเบราว์เซอร์
- ป้ายสถานะแมตช์ (ตามกำหนด สด จบเกม ...) ระดับผลคำทาย (สกอร์ตรง ผลต่างประตู ผลถูก พลาด) ป้ายกลุ่ม และชื่อรอบของสายน็อคเอาท์ (รวมแชมป์ / ชิงอันดับ 3) ตอนนี้ถูกแปลในทุกภาษา
- เหตุการณ์ในเกมของแมตช์จะไม่ยืดทั้งหน้าเมื่อมีเหตุการณ์จำนวนมากอีกต่อไป: บนจอที่กว้างขึ้น ไทม์ไลน์จะเลื่อนภายในพื้นที่จำกัด (บนมือถือหน้ายังเลื่อนตามปกติ)

- การสลับแท็บในหน้าแมตช์จะไม่กระโดดหน้ากลับไปด้านบนอีกต่อไป

## [1.0.2] - 2026-06-12

### เปลี่ยนแปลง
- Nostragoalus ตอนนี้ใช้ลิขสิทธิ์คู่แบบ **MIT OR WTFPL** - ใช้แบบใดก็ได้ที่คุณชอบ (ลิงก์ส่วนท้ายและหน้าลิขสิทธิ์ในแอปแสดงทั้งสองแบบ)
- ภาพหน้าจอโชว์เคสในหน้าแรกถูกอัปเดตเป็น UI ปัจจุบันและถ่ายทั้ง**โหมดสว่างและมืด** จัดให้ตรงกับธีมของคุณ ขนาดการ์ดสม่ำเสมอ (ไม่มีช่องว่างก่อนคำบรรยาย) เพิ่มมุมมองลีก และซ่อนเมนูผู้ดูแล / เครื่องมือนักพัฒนาในภาพ
- สูตร "คะแนนคำนวณอย่างไร" ในคำถามที่พบบ่อยตอนนี้ถูกแปลเป็นทั้งสี่ภาษา (เดิมเป็นภาษาอังกฤษเท่านั้น)
- รายการฉันทามติของบอตจะทำเครื่องหมายแมตช์ที่ยังไม่เริ่ม (เฉพาะผู้ดูแล) ด้วยตัวคั่นเดียวกับที่มุมมองคำทายของผู้เล่นใช้ แทนแบนเนอร์
- `mise run db-backup` ตัดดัมป์เก่าได้: `--keep N` / `BACKUP_KEEP` (N ล่าสุด) และ `--max-age-days D` / `BACKUP_MAX_AGE_DAYS` (ตามอายุ) ทั้งคู่ตั้งค่าเริ่มต้นเป็นไม่จำกัด จึงไม่มีการตัดเว้นแต่คุณเลือกใช้

### แก้ไข
- การ์ดดาวซัลโวแสดงคะแนนที่คำทายที่ชนะมีค่าแม้ล็อกแล้ว ไม่ใช่เฉพาะตอนกำลังเลือก
- ฟอร์มเข้าสู่ระบบจะส่งเมื่อกด Enter จากช่องใดก็ได้ (ตอนนี้เป็น `<form>` จริงแล้ว)

## [0.22.0] - 2026-06-11

### เพิ่ม
- อันดับชั่วคราวแบบสด: ขณะที่แมตช์กำลังแข่ง กระดานอันดับจะรวมคะแนนที่ผู้เล่นแต่ละคน*จะ*ได้ที่สกอร์ปัจจุบัน (ให้คะแนนเต็ม รวมโบนัส) จัดอันดับชั่วคราว แสดงส่วนต่าง "+N สด" ต่อแถวและป้าย LIVE ที่เต้นเป็นจังหวะ อัปเดตผ่าน WebSocket เมื่อสกอร์เปลี่ยน - ไม่มีการดึงข้อมูลเป็นช่วง ๆ
- ตารางคะแนนกลุ่มในหน้าแมตช์ตอนนี้ติดตามแมตช์สด: สกอร์ที่กำลังแข่งจะนับชั่วคราวและอัปเดตในที่
- คำทายดาวซัลโวของคุณตอนนี้แสดงบนโปรไฟล์ของคุณ (ธงทีม + 👟) ข้างแชมป์ของคุณ และการ์ดดาวซัลโวแสดงคะแนนที่คำทายที่ชนะมีค่า
- รายการตารางแข่งอัปเดตแบบสด - สถานะแมตช์และสกอร์รีเฟรชในที่ขณะเกมดำเนินไป ผ่าน WebSocket
- ขณะแมตช์สด สถิติและไทม์ไลน์เหตุการณ์จะอัปเดตอยู่เสมอ
- ผู้ดูแลที่ดูคำทายของผู้เล่นจะเห็นคำทายของแมตช์ที่ยังไม่เริ่มด้วย หลังตัวคั่นที่ทำเครื่องหมายว่าเฉพาะผู้ดูแล

### เปลี่ยนแปลง
- โบนัสความหายากจากคนทายถูกปรับใหม่ (สไตล์ MPP): ความหายากของสกอร์ที่ตรงเป๊ะของคำทายตอนนี้วัดในกลุ่มผู้เล่นที่ทาย*ผล*ถูก ไม่ใช่ทั้งสนาม และระดับแน่นขึ้น - เฉพาะคนส่วนน้อยที่ชัดเจนในกลุ่มที่ทายผลถูกเท่านั้นที่ได้โบนัส โดยไต่ชันขึ้นสำหรับการทายที่หายากจริง ๆ เกมที่ให้คะแนนไปแล้วจะถูกให้คะแนนใหม่ภายใต้กฎใหม่

### แก้ไข
- แมตช์ที่ FIFA ทำเครื่องหมายว่าจบเกมอาจค้างเป็น "สด" ในแอปได้นานถึงหนึ่งชั่วโมง (FIFA ตัดเกมที่จบแล้วออกจากฟีดสด) ตอนนี้เสียงนกหวีดหมดเวลาจะมาถึงในการดึงข้อมูลรอบ 2 นาทีถัดไป
- คุณจะแสดงบนกระดานอันดับที่คุณอยู่เสมอ แม้บัญชีของคุณจะซ่อนจากกระดานสาธารณะ - บัญชีที่ซ่อนไว้จะไม่เห็นกระดานว่างในลีกของตัวเองอีกต่อไป
- การอัปเดตสถิติ/เหตุการณ์สดจะแก้ในที่ แทนที่จะกะพริบตัวหมุนโหลดและวาดไทม์ไลน์ใหม่
- ชื่อลีกของบอตคนทายจะไม่ล้นส่วนหัวของหน้าบนมือถืออีกต่อไป
- โลโก้และชื่อแบนเนอร์ตอนนี้ลงตัวพร้อมกันเมื่อแถบเข้าที่ (ชื่อจะไม่หดก่อนดาวเคราะห์มากอีกต่อไป)

## [0.20.0] - 2026-06-11

### เพิ่ม
- กระดานอันดับตอนนี้แสดงคำทายดาวซัลโวของผู้เล่นแต่ละคน (ธงทีม + 👟) ข้างคำทายแชมป์ พร้อมชื่อทีมเต็ม / ชื่อผู้เล่นที่ทำให้เป็นมาตรฐานในทูลทิป
- ผู้ดูแล: การกระทำ "ใช้การเข้าร่วมอัตโนมัติทันที" จะเติมสมาชิกภาพลีก SSO ย้อนหลังให้ผู้ใช้ที่มีอยู่ทั้งหมดซึ่งโดเมนอีเมลถูกผู้ให้บริการที่เชื่อมไว้จับ (ไม่ต้องรอให้แต่ละคนเข้าสู่ระบบอีกครั้ง) บวกจำนวนผู้ใช้ทั้งหมด
- แถวผีของบอตฉันทามติจะปรากฏทันทีที่มีใครทาย (จัดอันดับสุดท้ายที่ 0 คะแนนก่อนให้คะแนน) ไม่ใช่เฉพาะเมื่อมีคะแนนแล้ว

### เปลี่ยนแปลง
- ราคาต่อรองจากเจ้ามือตอนนี้ปิดเป็นค่าเริ่มต้น (เลือกเปิด) การตั้งค่าและทูลทิปอธิบายราคา 1X2 และทศนิยม (1 = เจ้าบ้านชนะ, X = เสมอ, 2 = ทีมเยือนชนะ ยิ่งต่ำยิ่งมีโอกาส)
- หน้า "คำทายของฉัน" หายไปแล้ว - กล่องอันดับและการ์ดคำทายแชมป์ / ดาวซัลโวตอนนี้อยู่ในหน้าแมตช์ อันดับและจำนวนผู้เล่นจะอิงตามลีกที่เลือก
- ลูกศรการขยับบนกระดานอันดับตอนนี้สะท้อนเฉพาะการเปลี่ยนแปลงของคะแนน ไม่ใช่การเปลี่ยนรายชื่อสมาชิก (การเข้าร่วม การตั้งเป็นส่วนตัว หรือการถูกนำออกจะไม่ดันทุกคนอีกต่อไป)
- รูปหน้าผู้เล่นมาจาก UEFA สำหรับรายการ UEFA และ FIFA สำหรับฟุตบอลโลก รายการผู้เล่นดาวซัลโวจะแสดงกองหน้าก่อน

### แก้ไข
- รูปโปรไฟล์เสีย: รูปจากผู้ให้บริการยืนยันตัวตนที่ต้องใช้โทเค็น (เช่น Microsoft Graph) จะถูกดึงครั้งเดียวตอนเข้าสู่ระบบและฝังในตัว หรือแทนที่ด้วยรูปแทน แทนที่จะแสดงรูปที่เสีย
- สกอร์สดและยอดรวมคนทายเชื่อมต่อใหม่อัตโนมัติหลังเซิร์ฟเวอร์รีสตาร์ท แทนที่จะค้างอย่างเงียบ ๆ หน้าแมตช์ได้ป้ายลีกเพิ่ม
- การจัดแนวธงแชมป์ / ดาวซัลโว ดาวซ้ำของปุ่มโจ๊กเกอร์ โมดัลจัดการสมาชิกของผู้ดูแล (ชื่อยาว การแครช) และส่วนหัวกระดานอันดับล้นเมื่อชื่อลีกยาว

## [0.19.1] - 2026-06-10

### เพิ่ม
- ลีก: กลุ่มผู้เล่นที่จำกัดตามรายการแข่ง สร้างของคุณเอง (ส่วนตัวพร้อมโค้ดเข้าร่วมที่แชร์ได้ หรือสาธารณะให้ใครก็เข้าร่วมและดูอันดับได้) เข้าร่วมหลายลีกหรือไม่เลยก็ได้ - คำทายยังเป็นของคุณเสมอ ลีกเพียงกรองมุมมองเท่านั้น
- ป้ายลีกข้างปุ่มสลับรายการแข่งจะจดจำการเลือกล่าสุดของคุณต่อรายการแข่ง และจำกัดกระดานอันดับ (ตัวเลือกที่สามในปุ่มสลับขอบเขต) และยอดรวมคนทาย (ลีกก่อน รวมทุกคนหลังไอคอนลูกโลก - โบนัสความหายากยังคำนวณจากทุกคน)
- หน้า `/leagues`: จัดการลีกของคุณ (เปลี่ยนชื่อ คัดลอก/สร้างโค้ดใหม่ สลับสาธารณะ/ส่วนตัว เลื่อนผู้ดูแล โอนความเป็นเจ้าของ เตะ ออก ลบ) และเรียกดูลีกสาธารณะ `/leagues/:id` แสดงอันดับของลีก - สำหรับลีกสาธารณะแม้ไม่ได้เข้าร่วม
- บทบาทในลีก: เจ้าของและผู้ดูแลจัดการสมาชิกและโค้ดเข้าร่วม การออกหรือถูกเตะจะถูกจดจำ การเข้าร่วมอัตโนมัติของ SSO จึงไม่เคยเพิ่มใครกลับโดยขัดกับความต้องการ
- คำชี้แบบครั้งเดียวตอนเข้าสู่ระบบครั้งแรกโดยไม่มีลีก: กรอกโค้ดเข้าร่วม สร้างลีก เรียกดูลีกสาธารณะ หรือข้ามไปตลอด
- ผู้ดูแลจัดการทุกลีก (สร้าง อนุญาตให้ไม่มีเจ้าของ สมาชิก บทบาท การมองเห็น ลบ) และผู้ให้บริการ SSO สามารถเข้าร่วมผู้ใช้ที่จับตามโดเมนเข้าลีกที่เลือกอัตโนมัติทุกครั้งที่เข้าสู่ระบบ - ลีกเดียวกันผูกกับหลายผู้ให้บริการได้
- การตั้งค่าโปรไฟล์ส่วนตัว: ออกจากอันดับรวมและอันดับรายการแข่ง เฉพาะเพื่อนร่วมลีก (และผู้ดูแล) เท่านั้นที่เปิดโปรไฟล์ของคุณได้ ส่วนคนอื่นจะได้ 404 กระดานลีกยังจัดอันดับคุณให้เพื่อนร่วมสมาชิก และคำทายของคุณยังนับในยอดรวมคนทายแบบไม่ระบุตัวตน
- กระดานลีกแสดงลูกศรการขยับ (สแนปช็อตอันดับต่อลีก รีเฟรชพร้อมกับอันดับรวม) สำหรับสมาชิกและผู้ดูแล ยอดรวมคนทายของลีกอัปเดตแบบสดผ่าน WebSocket ส่งให้เฉพาะสมาชิกของลีกนั้น
- ผู้เล่นคนแรกที่เข้าร่วมลีกที่ไม่มีเจ้าของจะกลายเป็นเจ้าของ (ทั้งทางโค้ด สาธารณะ และการเข้าร่วมอัตโนมัติของ SSO) ผู้ดูแลยังมอบความเป็นเจ้าของให้ใครก็ได้ สมาชิกคนสุดท้ายที่ออกไปจะยังคงลีกไว้ - ว่างเปล่าและให้ผู้เข้าร่วมคนถัดไปรับช่วงได้ - และผู้ดูแลมีการกระทำ "ตัดลีกที่ว่างเปล่า" ที่ย้อนกลับไม่ได้เพื่อล้างลีกเหล่านั้น
- ความพยายามใส่โค้ดเข้าร่วมถูกจำกัดอัตรา (10 ครั้งต่อนาทีต่อบัญชี)

## [0.15.0] - 2026-06-09

### เพิ่ม

- ผู้ให้บริการ SSO สามารถจับหลายโดเมนอีเมล (คั่นด้วยจุลภาค รวมโดเมนย่อย) โดเมนที่ผู้ให้บริการรายอื่นจับไปแล้วจะถูกปฏิเสธ

- ฟอร์ม SSO ของผู้ดูแลจะแสดงทุกอย่างที่ฝั่ง IdP ต้องการขณะกรอก: OIDC redirect URI ขอบเขตและ claims, SAML ACS URL, SP entity ID และลิงก์ไปยัง SP metadata XML ที่สร้างขึ้น
- ผู้ดูแลซ่อนผู้ใช้คนใดจากกระดานอันดับได้ (ปุ่มสลับรูปตาในรายการผู้ใช้) ผู้เล่นที่ซ่อนไว้ยังเล่นต่อและยังนับในยอดรวมคนทาย บัญชี `create-admin` เริ่มต้นแบบซ่อนไว้
- การกด Enter หรือ space ในช่องกรอกสกอร์จะบันทึกและกระโดดเจ้าบ้าน → ทีมเยือน → แมตช์ถัดไป จึงพิมพ์ทั้งวันแข่งได้โดยไม่ใช้เมาส์

### แก้ไข
- การเข้าสู่ระบบด้วย SSO ที่อีเมลตรงกับบัญชีรหัสผ่านที่มีอยู่ตอนนี้จะเชื่อมกับบัญชีนั้น (รวมเป็นบัญชีเดียว) แทนที่จะล้มเหลวด้วย account_not_linked
- บัญชีที่จัดการโดย SSO (ไม่มีรหัสผ่านในเครื่อง) จะไม่เห็นหรือเข้าถึงการเปลี่ยนอีเมล รหัสผ่าน การจัดการ 2FA หรือ passkey อีกต่อไป - IdP เป็นเจ้าของสิ่งเหล่านั้น และบังคับใช้ฝั่งเซิร์ฟเวอร์ด้วย
- มือถือ: สูตรการให้คะแนนในคำถามที่พบบ่อยจะไม่บังคับให้เลื่อนแนวนอนทั้งหน้าอีกต่อไป (เลื่อนภายในกล่องของตัวเอง) และช่องว่างพื้นหลังหน้าระหว่างคำถามที่พบบ่อยที่เปิดอยู่กับคำตอบหายไปแล้ว
- มือถือ: แบนเนอร์ชื่อที่ปักหมุดเคยแสดงตัวอักษรยักษ์ที่ถูกครอปไม่กี่ตัว (แถบเดสก์ท็อปอัตราส่วน 19:1 ถูกครอปแบบ cover ให้พอดีความกว้างมือถือ) - แก้ไขด้วยการปรับแบนเนอร์ใหม่ในหัวข้อเปลี่ยนแปลง
- มือถือ: แถวการเปลี่ยนตัวจะไม่ล้นการ์ดไทม์ไลน์ของแมตช์อีกต่อไป แท็บฟอร์มจะคงสกอร์ไว้ในบรรทัดเดียวและซ่อนรายการแข่งไว้หลังวันที่ที่แตะได้ (ขีดเส้นใต้แบบจุด)
- เมนูนำทางส่วนหัวบนมือถือจะส่งสัญญาณการล้นแนวนอนด้วยขอบจาง ๆ และเลื่อนลิงก์ที่ใช้งานอยู่ให้เข้ามาในมุมมอง

### เปลี่ยนแปลง
- คำขอ API ที่กำลังทำงานอยู่จะถูกยกเลิกเมื่อออกจากหน้าหรือสลับรายการแข่ง (สัญญาณ vue-query ยอดรวมคนทาย และการดึงข้อมูลแบบ lazy ที่ช้าซึ่งอิง FIFA ของหน้าแมตช์)
- ผู้ให้บริการ SSO มีชื่อที่แสดงให้ผู้เล่นเห็น (ไม่บังคับ) SP metadata ของผู้ให้บริการ SAML ดาวน์โหลดได้ก่อนบันทึกผู้ให้บริการ (ฝั่ง IdP มักต้องใช้ก่อน) การขัดแย้งของการจับโดเมนตอนนี้ตรวจทั้งสองทิศทางของโดเมนย่อย
- ขั้นตอนลืมรหัสผ่าน: ลิงก์รีเซ็ตจากหน้าเข้าสู่ระบบ (เมลผ่าน SMTP) โดย better-auth จะสร้างรหัสผ่านในเครื่องตอนรีเซ็ต - นี่เป็นเส้นทางกู้คืนสำหรับผู้ใช้ที่ผู้ให้บริการ SSO ถูกลบด้วย บัญชีที่จัดการโดย SSO จะไม่ได้รับเมลรีเซ็ต
- การสมัครด้วยรหัสผ่านบนโดเมนที่ SSO จับไว้จะเตือนก่อน (พร้อมชื่อที่แสดงของผู้ให้บริการ) และต้องกด "ดำเนินการต่ออยู่ดี" อย่างชัดเจน
- เอนด์พอยต์จัดการผู้ให้บริการแบบ HTTP ของปลั๊กอิน SSO เอง (register/update/delete) ถูกบล็อก การจัดการผู้ให้บริการผ่าน admin API เท่านั้น
- "จัดการโดย SSO" ตอนนี้หมายถึงผู้ให้บริการที่ยังลงทะเบียนอยู่: การลบผู้ให้บริการจะปล่อยผู้ใช้ของมัน ซึ่งจะได้การจัดการข้อมูลรับรองคืนและตั้งรหัสผ่านได้ผ่านขั้นตอนรีเซ็ต
- การเข้าสู่ระบบด้วย SSO ที่สำเร็จจะลบรหัสผ่านในเครื่องของบัญชี (IdP กลายเป็นผู้มีอำนาจ) ขั้นตอนรีเซ็ตเป็นทางกลับหากผู้ให้บริการหายไป ผู้ดูแลยังเลิกเชื่อมผู้ใช้คนใดจาก SSO ได้
- รายการผู้ใช้ของผู้ดูแลย้ายการกระทำต่อผู้ใช้ไปไว้ในเมนูจุดสามจุด (เลื่อนบทบาท ซ่อน ถอด 2FA เลิกเชื่อม SSO แบน ลบ) พร้อมตัวบ่งชี้สถานะแบบอินไลน์
- ผู้ดูแลได้รับการยกเว้นจากการล้างรหัสผ่านของ SSO: รหัสผ่านของพวกเขาเป็นการเข้าถึงฉุกเฉินสำหรับลบผู้ให้บริการที่เสีย (ตัวสำรองสุดท้าย: `mise run create-admin` จากโฮสต์)
- เมื่อตั้งค่า SMTP การลบบัญชีจะยืนยันผ่านลิงก์ที่ส่งทางเมล (ใช้ได้กับบัญชี SSO ด้วย - ไม่มีการลบคลิกเดียวอีกต่อไป) หากไม่มี SMTP การยืนยันด้วยรหัสผ่าน / เซสชันใหม่ยังคงอยู่ ลิงก์ที่ส่งทางเมลแทนข้อกำหนด TOTP เพราะพิสูจน์ความเป็นเจ้าของกล่องจดหมาย
- แบนเนอร์หน้าแรกตอนนี้เป็น SVG แบบอินไลน์ชิ้นเดียวตลอดการเลื่อน โดยใช้ปุ่มปรับที่ขับด้วยการเลื่อนแทนการสลับภาพ: ดาวเคราะห์จะย่อขนาดในที่เพื่อให้อยู่ครบในแถบที่หดลง คำบรรยายรองจะจางหายไป และชื่อจะขยับลงเพื่อไม่ให้ถูกตัด การสลับระหว่างภาพแบนเนอร์สองแบบที่จัดองค์ประกอบต่างกันเคยดูเหมือนชื่อกระโดดไปด้านข้าง ตอนนี้ไม่มีการสลับใด ๆ

## [0.14.0] - 2026-06-08

### เพิ่ม

### แก้ไข
- แอสซิสต์ในแมตช์ UEFA เคยแสดงผู้รักษาประตูที่เสียประตูแทนผู้แอสซิสต์ (secondaryActor ของเหตุการณ์ประตูคือผู้รักษาประตู) แอสซิสต์จริงเป็นเหตุการณ์ ASSIST แยกต่างหาก ตอนนี้จับคู่กับประตูตามนาที จุดโทษจะแสดงว่าไม่มีแอสซิสต์อย่างถูกต้อง
- การทำเข้าประตูตัวเองของ UEFA ไม่เคยถูกตรวจพบ (ทำเครื่องหมายเป็น GOAL ที่มี subType 'OWN' ไม่ใช่ type OWN_GOAL) - บันทึกไว้ 0 ครั้งตลอดยูโร 2024 และแต่ละครั้งถูกให้เครดิตผิดให้ทีมของผู้ทำประตู ตอนนี้ตรวจพบแล้ว ให้เครดิตทีมที่ได้ประโยชน์ พร้อมแอสซิสต์ของผู้เล่นที่บังคับให้เกิด
- การนำเข้า/ซิงก์ของผู้ดูแลตอนนี้ล้างแคช query ของไคลเอนต์ รายการแข่งที่โหลดไว้ก่อน (เช่น ว่างเปล่า) จึงไม่แสดงข้อมูลเก่าค้างจนกว่าจะรีเฟรชเองอีกต่อไป

- ตัดการตั้งค่าที่ไม่ใช้ออก (NUXT_MATCH_PROVIDER, NUXT_FIFA_SEASON_ID, NUXT_WC_SEASON): ผู้ให้บริการและฤดูกาลเป็นแบบต่อรายการแข่ง (DB / API ฤดูกาล FIFA แบบสด) ตัวแปร env เหล่านี้ไม่เคยถูกอ่าน

### ความปลอดภัย / ปฏิบัติการ
- Postgres จะไม่เปิดพอร์ตโฮสต์ใน compose base ของโปรดักชันอีกต่อไป (แอปเข้าถึงผ่านในเครือข่าย) การเข้าถึงจากโฮสต์สำหรับ dev ในเครื่องถูกย้ายไป dev overlay ผูกกับ loopback แอปผูกกับ 127.0.0.1 (วางพร็อกซีย้อนกลับไว้ด้านหน้า)

- ฟอร์มแสดงผลห้านัดล่าสุดของแต่ละทีมในฟุตบอลนานาชาติทั้งหมด (รวมเกมกระชับมิตรและรอบคัดเลือก) พร้อมรายการแข่งและวันที่
- นัดถัดไปแสดงรายการเกมรายการแข่งของทีมหลังจากแมตช์ที่ดู - ผลจะแสดงแบบฟอร์มสำหรับเกมที่เล่นไปแล้ว
- การฉลองประตูสด: เมื่อสกอร์ของแมตช์สดเพิ่มขึ้น ภาพเคลื่อนไหวประตูมุมมองบุคคลที่หนึ่งสไตล์ pixel-art จะแสดงเป็นเวลาสามวินาที (ภาพที่มีผู้ร่วมสร้าง เคารพการลดการเคลื่อนไหว)
- วันที่ในหน้าแมตช์รวมปีด้วย (สถิติเจอกันย้อนหลังหลายทศวรรษ)

### เปลี่ยนแปลง
- สถิติเจอกัน ฟอร์ม และรายการการพบกันในระบบทั้งหมดจะตัดที่เวลาเริ่มแข่งของแมตช์ที่ดู - อนาคตไม่เคยกำหนดอดีต
- แท็บสถิติเจอกันจะแสดงเสมอ คู่ที่ไม่มีการพบกันบันทึกไว้จะได้หมายเหตุ "พบกันครั้งแรก" แทนที่แท็บจะหายไปเงียบ ๆ
- ประวัติคอมมิตทั้งหมดถูกเขียนใหม่เป็นตัวตน Arzaroth

### แก้ไข
- สายน็อคเอาท์เคยจัดแมตช์ป้อนไว้ใต้ตัวแม่ที่ผิด (FIFA แสดงรายการแมตช์น็อคเอาท์อย่างไม่เป็นระเบียบ โมร็อกโกและบราซิลอยู่ผิดฝั่ง) ตอนนี้การจัดลำดับร่วมกันจะไล่ลงจากรอบชิงสำหรับทุกผู้ให้บริการ
- การ์ดสายน็อคเอาท์เคยแสดงสกอร์จุดโทษ (0) ในแมตช์ที่ตัดสินในเวลาปกติ ตอนนี้จุดโทษจะแสดงเป็นตัวยกเฉพาะการดวลจุดโทษจริงเท่านั้น
- รอบชิงถูกตรึงไว้ที่เส้นกลางของรอบรองชนะเลิศ เส้นเชื่อมจะรวมกันกลางช่องว่างและนำตรงเข้าสู่แมตช์ถัดไป วันที่จัดกึ่งกลางในทุกการ์ด
- การ์ดสายน็อคเอาท์ยูโรเคยแสดง "Invalid Date" (แมตช์สายน็อคเอาท์ UEFA ไม่มีเวลาเริ่มแข่ง)
- แท็บผู้เล่นของแมตช์แสดงเฉพาะผู้ที่มีส่วนร่วม แทนรายชื่อ 26 คนเต็มที่เป็นศูนย์ทั้งหมด
- หน้าเกี่ยวกับ: เครื่องหมายของ Motion เองแทนโลโก้เครื่องมือออกแบบ Framer favicon ทางการสำหรับ Nuxt I18n, node-postgres, Nodemailer, maildev
